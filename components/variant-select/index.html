VariantSelect

VariantSelect component

The VariantSelect component gives you a selection dropdown component for a page's variants.

Options:

In addition to the Dropdown options, the following options are:

  • root: the path to the variant's root
  • location: the current page's path

Example:

<VariantSelect
   overlay={true}
   direction="se"
   menuWidth="min(30ch, 500px)"
   root="/app/version-two"
   location="/app-version-two/usage" />
